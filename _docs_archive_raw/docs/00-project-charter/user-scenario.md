# Пользовательский сценарий приложения
Пользователь запускает приложение и попадает на экран приветствия, где может выбрать между регистрацией нового аккаунта и входом в существующий. При регистрации необходимо указать адрес электронной почты, придумать пароль и ввести желаемое отображаемое имя. После успешной аутентификации пользователь автоматически переходит на главный экран приложения, который представляет собой вкладку **«Мои образы»** с демонстрацией созданных ранее комплектов одежды.

Нижняя панель навигации содержит три основные вкладки: **«Мои вещи»**, **«Мои образы»** и **«Настройки»**. На вкладке **«Мои вещи»** отображается сетка или список всех добавленных предметов гардероба с возможностью поиска по названию и фильтрации по категориям или сезонам. Для добавления новой вещи пользователь нажимает кнопку создания, после чего открывается форма, где необходимо загрузить фотографию предмета, указать его название, при желании добавить описание, выбрать подходящую категорию из предопределённого списка и отметить сезонность. После сохранения вещь появляется в общем списке и становится доступной для просмотра, редактирования и использования при составлении образов.

При переходе на вкладку **«Мои образы»** пользователь видит коллекцию созданных ранее комплектов одежды. Для создания нового образа нажимается соответствующая кнопка, и открывается конструктор, где представлены специальные слоты для различных категорий одежды: головной убор, верхняя часть гардероба, нижняя часть, обувь, а также дополнительные слоты для верхней одежды и аксессуаров. В каждый слот можно добавить одну из ранее загруженных вещей путём выбора из списка доступных предметов подходящей категории. После комплектации всех необходимых элементов пользователь может задать название образа, добавить описание, отметить сезонность и сохранить результат. Созданный образ отображается в общей коллекции и может быть просмотрен в деталях, отредактирован или удалён.

Вкладка **«Настройки»** содержит параметры персонализации приложения, включая переключение между светлой и тёмной темой оформления, выбор языка интерфейса и настройку уведомлений. Отсюда же доступен переход к управлению учётной записью, где пользователь может изменить свой аватар, отредактировать отображаемое имя, сменить пароль, обновить адрес электронной почты. При необходимости пользователь может выйти из учётной записи или полностью удалить свой аккаунт со всеми связанными данными.

Приложение поддерживает работу в офлайн-режиме: все добавленные вещи и созданные образы сохраняются в локальном хранилище устройства и автоматически синхронизируются с сервером при восстановлении интернет-соединения, что обеспечивает бесперебойный доступ к гардеробу в любых условиях.
