## Начало работы

### Системные требования

- **ОС:** Android 9 (API 28) и выше
- **Память:** минимум 100 МБ свободного места
- **Интернет:** требуется для синхронизации данных
- **Подключение:** Wi-Fi или мобильная сеть

### Установка приложения

1. Соберите APK из исходников (директория myshelf-app)
2. Откройте приложение — вы попадёте на экран авторизации
3. Зарегистрируйте новый аккаунт или войдите, если у вас уже есть учётная запись

## Аутентификация

### Регистрация нового аккаунта

![Экран авторизации](../08-ui/screenshots/0_auth.jpg)

**Шаги:**

1. На экране авторизации нажмите **"Создать аккаунт"**
2. Введите следующие данные:
   - **Электронная почта** — будет использоваться для входа
   - **Пароль** — минимум 8 символов, содержащий буквы и цифры
   - **Подтверждение пароля** — повторите пароль
   - **Имя** (опционально) — для персонализации приложения

3. Нажмите **"Зарегистрироваться"**
4. Система отправит письмо подтверждения на вашу электронную почту
5. Подтвердите адрес и вернитесь в приложение

⚠️ **Важно:** 
- Пароль хранится в защифрованном виде
- Не забывайте свой пароль — восстановление происходит через электронную почту
- Используйте надёжный пароль, не совпадающий с другими сервисами

### Вход в аккаунт

1. На экране авторизации введите:
   - **Email** — ваша электронная почта
   - **Пароль** — ваш пароль
2. Нажмите **"Войти"**
3. Приложение загрузит ваши данные и отправит вас на главный экран

### Восстановление пароля

1. На экране авторизации нажмите **"Забыли пароль?"**
2. Введите вашу электронную почту
3. Проверьте письмо со ссылкой восстановления
4. Установите новый пароль
5. Вернитесь в приложение и войдите с новым паролем

## Основной интерфейс

Приложение состоит из **трёх основных разделов**, доступных через нижнее меню:

| Раздел | Значок | Функция |
|--------|--------|---------|
| 👗 **Мои вещи** | Вешалка | Каталог всей одежды и аксессуаров |
| 👔 **Мои образы** | Карточки | Готовые комплекты одежды |
| ⚙️ **Настройки** | Шестеренка | Параметры аккаунта и приложения |

### Поддержка светлой и тёмной темы

Приложение автоматически определяет тему вашего устройства, но вы можете изменить её вручную:

1. Перейдите в **Настройки** → **Оформление**
2. Выберите **Светлая тема**, **Тёмная тема** или **По умолчанию (система)**
3. Интерфейс сразу же обновится

## Управление вещами

### 1. Просмотр каталога вещей

![Экран "Мои вещи"](../08-ui/screenshots/4_items.jpg)

В этом разделе отображается **весь ваш гардероб**. Вещи организованы в виде карточек с:
- Фотографией
- Названием
- Категорией (платье, рубашка, штаны и т.д.)
- Сезоном (лето, зима, межсезонье)

**Функции:**
- **Поиск** — используйте строку поиска для быстрого поиска вещей
- **Фильтрация** — нажмите на категорию для фильтрации по типу
- **Сортировка** — сортируйте по дате добавления, названию или сезону

### 2. Просмотр деталей вещи

![Экран деталей вещи](../08-ui/screenshots/5_item_details.jpg)

Нажмите на карточку вещи, чтобы увидеть полную информацию:

- **Фотография** — основное изображение вещи
- **Название** — название предмета
- **Описание** — детальное описание (материал, размер, состояние)
- **Категория** — тип одежды
- **Сезон** — когда её можно носить

**Доступные действия:**
- **Редактировать** — изменить информацию о вещи
- **Удалить** — удалить вещь из каталога

### 3. Добавление новой вещи

![Экран добавления вещи](../08-ui/screenshots/6_item_create.jpg)

Для добавления вещи в каталог:

1. На экране **"Мои вещи"** нажмите **кнопку "+** (плюс внизу)
2. Нажмите **"Добавить вещь"**
3. Заполните поля:
   - **Фотография** — сделайте снимок или выберите из галереи
   - **Название** — например, "Синяя рубашка"
   - **Описание** — опишите вещь подробнее
   - **Категория** — выберите тип одежды из списка
   - **Сезон** — выберите, когда её носить

4. Нажмите **"Сохранить"**

### 4. Редактирование вещи

1. Откройте детали вещи
2. Нажмите **"Редактировать"**
3. Измените нужные поля
4. Нажмите **"Сохранить"**

### 5. Удаление вещи

1. Откройте детали вещи
2. Нажмите **"Удалить"**
3. Подтвердите удаление в диалоговом окне

⚠️ **Внимание:** После удаления вещь будет удалена со всех образов!

## Управление образами

### 1. Просмотр ваших образов

![Экран "Мои образы"](../08-ui/screenshots/1_outfits.jpg)

В этом разделе хранятся **готовые комплекты одежды** — комбинации вещей, которые вы собрали вместе.

Каждый образ отображается в виде карточки с:
- **Коллажем фотографий** — фотографии всех вещей в комплекте
- **Названием** — название образа
- **Сезоном** — когда его можно носить
- **Датой создания** — когда был создан этот комплект

**Функции:**
- **Фильтрация по сезону** — показать только летние, зимние или межсезонные образы
- **Поиск** — найти образ по названию
- **Избранное** — отметьте свои любимые комплекты звёздочкой

### 2. Просмотр деталей образа

![Экран деталей образа](../08-ui/screenshots/2_outfit_details.jpg)

Нажмите на карточку образа, чтобы увидеть его в полном размере:

- **Визуальное представление** — как выглядит комплект вместе
- **Список вещей** — какие предметы входят в комплект
- **Описание** — заметки о стиле (например, "Для офиса", "На прогулку")
- **Сезон** — когда его рекомендуется носить

**Доступные действия:**
- **Редактировать** — изменить состав комплекта
- **Удалить** — удалить образ (вещи останутся в каталоге)
- **Поделиться** — скопировать ссылку или поделиться через соцсети
- **Сохранить** — добавить в избранное

### 3. Создание нового образа

![Экран создания образа](../08-ui/screenshots/3_outfit_create.jpg)

Для создания нового комплекта:

1. На экране **"Мои образы"** нажмите **кнопку "+** (плюс внизу)
2. Нажмите **"Создать образ"**
3. Экран разделён на **слоты** для разных категорий:

   | Слот | Категория | Что добавить |
   |------|-----------|-------------|
   | 🔝 | Верхняя одежда | Куртка, кардиган, пиджак |
   | 👕 | Топ | Рубашка, блузка, футболка |
   | 👖 | Низ | Штаны, юбка, шорты |
   | 👟 | Обувь | Туфли, кроссовки, сапоги |
   | 🧣 | Аксессуары | Сумка, шарф, украшения |

4. Для каждого слота:
   - Нажмите на слот
   - Выберите вещь из каталога
   - Вещь появится в слоте

5. После выбора всех вещей введите:
   - **Название** — например, "Офисный стиль"
   - **Описание** — добавьте заметки
   - **Сезон** — когда носить этот комплект

6. Нажмите **"Сохранить"**

### 4. Редактирование образа

1. Откройте детали образа
2. Нажмите **"Редактировать"**
3. Измените слоты (нажмите на слот для замены вещи)
4. Измените название, описание или сезон
5. Нажмите **"Сохранить"**

### 5. Удаление образа

1. Откройте детали образа
2. Нажмите **"Удалить"**
3. Подтвердите удаление

## FAQ

### ❓ Часто задаваемые вопросы

#### 1. Могу ли я использовать приложение без интернета?

Да! После первой синхронизации приложение загружает все данные локально. Вы можете:
- Просматривать вещи и образы
- Просматривать детали
- Редактировать существующие данные

Новые данные будут синхронизированы при подключении к интернету.

#### 2. Как восстановить случайно удалённую вещь?

К сожалению, удаление необратимо. Рекомендуется:
- Регулярно экспортировать данные (Настройки → Экспорт)
- Восстановить из резервной копии

#### 3. Есть ли ограничение на количество вещей?

Нет жёстких ограничений, но рекомендуется:
- Не более 5 МБ на одну фотографию
- До 10,000 вещей на аккаунт

#### 4. Как изменить категорию вещи?

1. Откройте детали вещи
2. Нажмите **"Редактировать"**
3. Измените категорию
4. Сохраните

#### 5. Могу ли я создать собственную категорию?

На данный момент доступны только предустановленные категории.

#### 6. Что делать, если образ не сохранился?

Проверьте:
- Введено ли название образа
- Выбраны ли хотя бы основные вещи (верх, низ)
- Достаточно ли памяти на устройстве

## Решение проблем

### 🔧 Типичные проблемы и их решения

#### Приложение не запускается

**Решение:**
1. Убедитесь, что у вас установлена Android 9 (API 28) или выше
2. Очистите кэш приложения:
   - Перейдите в **Параметры → Приложения → myShelf → Хранилище → Очистить кэш**
3. Переустановите приложение

#### Не могу войти в аккаунт

**Решение:**
1. Проверьте правильность электронной почты и пароля
2. Нажмите **"Забыли пароль?"** и сбросьте пароль
3. Проверьте подключение к интернету

#### Вещи не синхронизируются

**Решение:**
1. Проверьте подключение к интернету
2. Перейдите в **Настройки → Синхронизация** и нажмите **"Синхронизировать сейчас"**
3. Если это не помогло, очистите кэш и переавторизуйтесь

#### Фотография не загружается

**Решение:**
1. Проверьте размер файла (не более 5 МБ)
2. Используйте форматы JPG или PNG
3. Проверьте, достаточно ли памяти на устройстве
4. Перезагрузите приложение

#### Приложение работает медленно

**Решение:**
1. Очистите кэш приложения
2. Удалите неиспользуемые вещи и образы
3. Выключите фоновую синхронизацию, если она не нужна
4. Перезагрузите устройство
