# Progile Mobile

Современное мобильное приложение для логистической системы мониторинга транспорта.

## 📱 О приложении

Progile Mobile — это приложение для водителей, диспетчеров и логистов, которое предоставляет:

- 🚚 Отслеживание транспорта в реальном времени
- 🗺 Просмотр маршрутов и контрольных точек
- 📊 Аналитику и отчёты
- 🔔 Систему уведомлений
- 💬 Встроенный чат для коммуникации
- 📈 KPI метрики и дашборды

## 🎨 Дизайн

Приложение выполнено в стиле **Material Design 3** с использованием:

- Светлая тема по умолчанию
- Мягкие тени и скругления 16-24px
- Плавные анимации
- Современный минималистичный UI
- Адаптивный дизайн для мобильных устройств

### Цветовая схема

- Primary: #4DA6FF (синий)
- Background: #F8FBFF (светло-голубой)
- Success: #22C55E (зелёный)
- Warning: #F59E0B (оранжевый)
- Danger: #EF4444 (красный)

## 🏗 Структура приложения

### Экраны

1. **Dashboard (Главная)** - KPI метрики, карта, уведомления
2. **Transport (Транспорт)** - Список транспорта с поиском
3. **Routes (Маршруты)** - Активные и завершённые маршруты
4. **Reports (Отчёты)** - Аналитика и экспорт данных
5. **Profile (Профиль)** - Настройки пользователя

### Компоненты

- `TransportCard` - Карточка транспорта
- `RouteCard` - Карточка маршрута
- `KPIWidget` - Виджет метрики
- `MapWidget` - Интерактивная карта
- `AlertCard` - Карточка уведомления
- `StatusBadge` - Бейдж статуса
- `BottomNav` - Нижняя навигация
- `AppBar` - Верхняя панель
- `NotificationCenter` - Центр уведомлений

## 🛠 Технологии

- **React 18.3.1** - UI библиотека
- **TypeScript** - Типизация
- **Tailwind CSS v4** - Стилизация
- **Vite** - Сборщик
- **pnpm** - Менеджер пакетов

## 📦 Установка и запуск

```bash
# Установка зависимостей
pnpm install --ignore-scripts

# Запуск dev сервера
pnpm run dev

# Production-сборка
pnpm run build
```

Локальный URL после запуска: `http://localhost:5173`.

## ✅ Реализовано в текущей версии

- Авторизация и регистрация в демо-контуре с ролями: диспетчер, логист, администратор, руководитель, водитель.
- Единое состояние приложения с сохранением в `localStorage` как оффлайн-кэш.
- CRUD для транспортных средств: добавление, редактирование, удаление, поиск и фильтрация по статусу.
- CRUD для маршрутов: создание, редактирование, удаление, назначение транспорта, изменение статуса.
- Детальные экраны транспорта и маршрутов с live-показателями, контрольными точками и чат-комментариями.
- Дашборд с KPI, уведомлениями и имитацией обновления GPS-данных.
- Отчёты с графиками, экспортом PDF/XLS/CSV и перечнем REST endpoints для Spring Boot backend.
- Профиль пользователя, настройки уведомлений, тёмная тема и выход из аккаунта.

## 🎯 Основные функции

### Dashboard
- Отображение KPI метрик
- Live статус транспорта
- Интерактивная карта
- Лента уведомлений

### Транспорт
- Поиск по номеру, водителю, модели
- Фильтрация по статусу
- Детальная информация о ТС
- История движения и остановок
- График скорости
- Чат с водителем

### Маршруты
- Фильтры: Все / Активные / Запланированные / Завершённые
- Прогресс выполнения
- Контрольные точки
- ETA (расчётное время прибытия)
- Уведомления об отклонениях

### Отчёты
- Графики выполнения маршрутов
- Анализ скорости
- Топ водителей
- Экспорт в PDF, Excel, CSV

### Профиль
- Информация пользователя
- Настройки уведомлений
- Переключатель тёмной темы
- Статистика работы
