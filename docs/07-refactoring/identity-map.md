# Identity Map

На backend JPA persistence context уже выполняет роль identity map для managed entities. На Android `ProgileViewModel` хранит актуальные списки транспорта и маршрутов, а `DemoTransportRepository` предоставляет локальный снимок для офлайн-демонстрации.

Правило: если объект транспорта уже есть в состоянии приложения, экран деталей должен брать его из общего состояния, а не делать повторный запрос без необходимости.
