# Расчет ROI

| Показатель | Значение |
|---|---:|
| Разработка и внедрение | 650 000 руб. |
| Сопровождение за год | 120 000 руб. |
| Совокупный эффект в месяц | 95 000 руб. |
| Совокупный эффект в год | 1 140 000 руб. |

```text
ROI = ((1 140 000 - 770 000) / 770 000) * 100% = 48.1%
Срок окупаемости = 770 000 / 95 000 = 8.1 месяца
```

Проект экономически оправдан для подразделения, где диспетчеризация и контроль маршрутов выполняются вручную или через разрозненные каналы.
