# SWOT-анализ

| Сторона | Факторы |
|---|---|
| Strengths | мобильность, единая точка контроля, JWT, offline cache, Swagger |
| Weaknesses | часть доменной логики демонстрационная, нужен правильный backend URL |
| Opportunities | карты, GPS-сервисы, уведомления, расширенная аналитика, экспорт |
| Threats | нестабильная сеть, ошибки ручного ввода, требования безопасности |

Вывод: первая версия закрывает учебные и базовые бизнес-требования. Главный риск реальной эксплуатации - надежность данных и сети, поэтому в архитектуру включен кэш.
