# Скриншоты интерфейса Progile

В папке `screenshots` сохранены изображения фактического транспортного интерфейса, использованного при реализации Kotlin-прототипа.

| Файл | Содержание |
|---|---|
| `dashboard-screen.png` | KPI, статусы и карта транспорта |
| `transport-screen.png` | поиск и карточки транспортных средств |
| `vehicle-detail-screen.png` | сведения об автомобиле и водителе |
| `routes-screen.png` | активные маршруты и прогресс |
| `reports-screen.png` | сводка и недельная диаграмма |
| `profile-screen.png` | профиль диспетчера и настройки |

Запуск клиента:

```powershell
cd android
.\gradlew.bat :app:installDebug
```
