# 08. Мобильный UI

Интерфейс реализован на Kotlin с Jetpack Compose и Material 3. Приложение содержит семь самостоятельных экранов и нижнюю навигацию.

| Экран | Kotlin-файл | Назначение |
|---|---|---|
| Вход и регистрация | `AuthScreen.kt` | локальная регистрация и аутентификация |
| Главная | `HomeScreen.kt` | KPI, статусы и OpenStreetMap |
| Транспорт | `TransportScreen.kt` | поиск и список транспортных средств |
| Детали ТС | `VehicleDetailsScreen.kt` | карта, водитель и показатели рейса |
| Маршруты | `RoutesScreen.kt` | фильтрация и прогресс маршрутов |
| Отчёты | `ReportsScreen.kt` | недельная сводка и диаграмма |
| Профиль | `ProfileScreen.kt` | настройки, тема, язык и выход |

Скриншоты находятся в каталоге [screenshots](screenshots), описание — в [screenshots.md](screenshots.md).
