# Пояснительная записка

## Тема

Разработка мобильного приложения Progile для мониторинга транспорта.

## Цель

Создать мобильный прототип и серверную часть для контроля транспортных средств, маршрутов, GPS-состояний, отчётов и ролей пользователей.

## Технологии

- Kotlin, Jetpack Compose, Material 3, MVVM и StateFlow;
- SharedPreferences, локальная регистрация и DemoTransportRepository;
- WebView/OpenStreetMap с Canvas-fallback;
- Java 17, Spring Boot 3, Spring Security и JWT;
- PostgreSQL, Spring Data JPA и OpenAPI;
- JUnit, MockMvc и JaCoCo.

## Результаты

1. Реализовано семь Android-экранов и нижняя навигация.
2. Добавлены вход, локальная регистрация, сохранение сессии и разграничение демонстрационных данных.
3. Реализована интерактивная карта и офлайн-заглушка.
4. Реализован Spring Boot REST API с JWT и обязательным CRUD `/api/entities`.
5. Добавлены endpoints транспорта, маршрутов, GPS, отчётов и комментариев.
6. Подготовлены OpenAPI, Postman collection, JUnit-тесты, APK и JAR.

Полная оформленная работа находится в `12-final-report/Курсовой_проект_Progile.docx` и в папке `artifacts` корня проекта.
