# Результаты тестирования backend

Последний сохранённый запуск содержит 15 тестов в пяти классах: ошибок, падений и пропусков нет. Исходные тесты находятся в `backend/src/test/java/com/progile/backend`.

Полный HTML-отчёт JaCoCo создаётся командой `mvn verify` в `backend/target/site/jacoco`. В архив не включён старый отчёт `com.myshelf.wardrobe`, так как он относится к другому проекту.
