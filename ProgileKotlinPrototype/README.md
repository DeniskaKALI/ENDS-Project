# Progile Kotlin Prototype

Нативный Android-прототип системы мониторинга транспорта. Проект написан на Kotlin и Jetpack Compose и открывается в Android Studio как самостоятельное приложение.

## Что реализовано

- 6 экранов: главная, транспорт, карточка автомобиля, маршруты, отчёты и профиль;
- вход, локальная регистрация и сохранение пользовательской сессии;
- нижняя навигация между основными разделами;
- поиск по номеру, водителю и модели транспорта;
- фильтрация маршрутов по статусу;
- интерактивные вкладки отчётов и карточки автомобиля;
- рабочие переключатели уведомлений и тёмной темы;
- смена языка профиля и диалог политики конфиденциальности;
- интерактивная карта OpenStreetMap и диаграмма выполнения маршрутов;
- архитектура MVVM: `ViewModel`, `StateFlow`, модели и репозиторий;
- автономная работа без сервера; при отсутствии интернета карта переключается на локальную схему.

## Запуск в Android Studio

1. Откройте Android Studio.
2. Нажмите **Open** и выберите папку `ProgileKotlinPrototype`.
3. Дождитесь завершения Gradle Sync.
4. Выберите эмулятор Android API 26 или новее либо подключённый телефон.
5. Нажмите зелёную кнопку **Run app**.

Сначала откроется экран входа. Сервер для демонстрации не требуется. Для загрузки реальной карты OpenStreetMap нужен интернет, остальные экраны работают офлайн.

## Готовый APK

После сборки Android Studio создаёт APK по пути:

`app/build/outputs/apk/debug/app-debug.apk`

Этот файл можно установить на Android-устройство. На телефоне должна быть разрешена установка приложений из выбранного файлового менеджера.
